Ceci est un patch non-officielle permettant d'offrir une fan traduction française
au jeu I HAVE NO MOUTH AND I MUST SCREAM.
Ce patch n'est pas approuvé ni par Cyberdreams, ni par The Dreamers Guild, ni par QUByte Interactive,
ni par Acclaim Entertainment, ni par Nightdive Studios, ni par DotEmu.

Il est prévu pour fonctionner sur les versions anglaise du jeu, et a été testé avec succès sur la
version publié sur Steam par Nightdive Studios (mais devrais fonctionner également pour d'autres)

Le patch doit cibler le dossier contenant SCRIPTS.RES et SCREAM.RES de la version anglaise du jeu.
(les fichiers S.prg ou S.EXE seront patché également si présent dans le dossier cible)

Dans les distributions modernes du jeu avec menu et toutes les langues installés, le répertoire à cibler est :
(dossier d'installation de IHNMAIMS)\IHNMAIMS_Data\StreamingAssets\disk_EN\scream

Pour cibler la version ScummVM sur Steam (vérifier bien que l'installation Steam est alors réglé sur anglais):
(dossier d'installation de IHNMAIMS)\Classic

(Sur Steam vous pouvez passer par "Gérer > Parcourir les fichiers locaux", pour trouver rapidement le dossier d'installation).

Vous devrez mettre la langue sur anglais dans le menu moderne du jeu afin que la version patché en FR soit utilisée.
